Steam Market Intelligence Dashboard
Project Overview

This project is an end-to-end data analytics portfolio project focused on analyzing the Steam game marketplace.
The objective is to collect, process, store, and visualize Steam marketplace data to generate business insights related to:

Customer review trends
Free vs paid game behavior
Pricing analysis
Market segmentation
Product recommendation insights

The project simulates the role of a Data Analyst working in a gaming market intelligence environment.

Business Objective

The purpose of this project is to understand how games are positioned in the Steam marketplace by analyzing:

Customer sentiment through review distribution
Market pricing behavior
Premium vs free-to-play ecosystem
Top-performing and highly reviewed games

The dashboard is designed to help stakeholders quickly understand customer preferences and market patterns.

Workflow Steps

Clean and transform raw data using Pandas
Store cleaned data in SQL Server
Perform SQL analysis queries
Build interactive dashboard in Power BI

Several games consistently receive strong community reception
Free-to-play games remain important in multiplayer/live-service categories
Game pricing distribution is highly skewed due to premium AAA titles
Customer sentiment can be used as a proxy for market popularity
